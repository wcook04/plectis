# Anti-Principles

These are explicit failure shapes this slice rejects.

- A projection row claims to be source authority.
- A passing binding validator claims public leaf readiness.
- A source capsule includes a body instead of redacted metadata.
- Duplicate pattern ids are accepted without deterministic rejection.
- An unsupported authority-chain handle is treated as authority.
- A fixture copies forbidden content bodies instead of using synthetic data.
- A generated receipt is edited by hand.
