Provenance

Plectis is an independent, AI-assisted project by Will Cook. The public release artifact is the standalone source repository, not the private working root behind it. Microcosm remains a legacy and technical compatibility name for repo paths, package ids, generated state, and historical links.

Lineage

Microcosm became Plectis on 21 June 2026 after a public naming check found an established Southampton hypermedia system with the same name. The acknowledgement is source-backed and does not imply endorsement or affiliation.

Site contract

The site is a static projection over public source refs. It has no runtime backend and no browser path back into the private working environment. It is not legal, financial, investment, trading, security, medical, or other professional advice.

The repository is the source of record. This site is a map over it, and when the two disagree, the repository wins.

Every page is generated from the source files listed above and the component index is a projection of them. The current content fingerprint is ea832840962f132bdf6f89fbb3401235; when the source changes, a rebuild regenerates the pages and a drift check fails until the site is current.
