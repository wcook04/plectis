# Work Landing Control Spine Bundle

This bundle copies the non-secret macro work-landing control-plane source into Microcosm for public inspection and validation. It carries the command wrapper, status/reconcile model, mission transaction preflight wrapper, preflight kernel, and private-index scoped commit actuator source.

The validator treats these files as source-open substrate, not as live actuator authority. Receipts carry refs, digests, counts, anchor checks, authority ceilings, and secret-exclusion results; they do not inline source bodies.
