## Routing Model

The ledger rows are raw ore. Future reconstruction should select organs: groups of patterns that share a fixture, proof gate, source boundary, and cold-start story. A pattern is strong when it changes the substrate metabolism: work, status, navigation, doctrine, evidence, validation, continuity, agent behavior, formal proof, or self-improvement.

The leaf bar should be:

- it can run or validate on a small synthetic fixture;
- it exposes its source boundary and anti-claims;
- it has enough root substrate to make sense from cold start;
- it is not just a present UI, release packet, compatibility shim, or private-root view;
- it composes with sibling patterns instead of hiding them as unexplained infrastructure.
