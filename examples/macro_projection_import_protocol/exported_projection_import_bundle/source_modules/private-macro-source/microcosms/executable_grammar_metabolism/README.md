# Executable Grammar Metabolism

This specimen is a public-safe toy analogue for treating release standards as executable grammar.
It is not the private standards engine, not a private candidate corpus, and not a publication claim.

Run it from the release root:

```bash
PYTHONPATH=src python3 -m idea_microcosm.cli build-executable-grammar-metabolism-specimen --root . --write-receipt
PYTHONPATH=src python3 -m idea_microcosm.cli validate --root .
```

The fixture mutates one public-safe release-candidate row and records the rule that accepts, blocks, or repairs the result.
Each decision now routes to a bounded worker action and transaction status instead of ending at a loose validation dump.
It also reads the provider harness canary board and turns each synthetic provider status-channel row into a replay repair route with owner lane, restart point, and local-only transaction status.
Each grammar and provider replay decision carries `source_capsule_provenance`: a hashed public-safe source clip, semantic carryforward fields, worker action refs, and anti-claims.

The boundary is fail-closed: no private registry, raw operator note, private standard engine, real provider transcript, website publication claim, or clone-ready claim is included.
