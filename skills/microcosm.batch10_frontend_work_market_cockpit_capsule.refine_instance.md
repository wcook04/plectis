## Authority Boundary

This skill governs bounded Batch 10 frontend Work/Market cockpit instance refinement only. It does not grant live UI correctness, browser or provider access, Task Ledger or Work Ledger mutation authority, market advice, trading or prediction authority, source mutation, publication, release authority, private-root equivalence, or whole-system correctness.

Anti-claims:

- Digest repair is not source mutation authority.
- Negative-case coverage is not live UI proof.
- Receipt repair cannot authorize market advice, publication, or release.
