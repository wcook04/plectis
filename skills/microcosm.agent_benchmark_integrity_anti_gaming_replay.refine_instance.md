## Authority Boundary

This skill governs bounded instance repair only. It cannot widen the benchmark-integrity replay authority ceiling, turn trace fixtures into benchmark scores, or treat body-free receipts as provider execution evidence.

Anti-claims:

- Correcting an instance edge is not proof that the replay is complete.
- Negative-case coverage is not propagation proof.
- Projection parity does not authorize public benchmark or release claims.
