## Authority Boundary

This skill source governs bounded anti-principle-instance correction only. It does not authorize source-of-record migration, new active anti-principle creation, or broad failed-principle wiring.

Anti-claims:

- Count movement is not proof that every failure mode has a negated principle.
- A guard edge is not a positive support edge.
- Generated health can expose the gap, but source evidence or validator computation must close it.
