## Authority Boundary

This skill governs public agent-trace authoring only. It does not certify private transcript contents, live session behavior, provider execution, release readiness, or whole-system correctness.

Anti-claims:

- Manifest ids are routing evidence, not source-body evidence.
- Route-observability receipts are not private trace access.
- Generated projections are not proof that an agent trace is complete.
