## Authority Boundary

This skill governs bounded MCP tool authority replay repair only. It cannot widen the authority ceiling, turn public fixtures into live tool evidence, or treat receipts as account-safety proof.

Anti-claims:

- Correcting an instance edge is not proof that tool-authority coverage is complete.
- Negative-case coverage is not propagation proof.
- Projection parity does not authorize live tool or release claims.
