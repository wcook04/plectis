## Authority Boundary

This skill source governs bounded runtime organ instance correction only. It does not authorize release, provider calls, live repository mutation outside the claimed lane, private body export, hidden fixture disclosure, benchmark scores, theorem proof, or whole-organ correctness claims.

Anti-claims:

- Correcting a source ref, fixture ref, receipt ref, or authority ceiling is not evidence that an organ is complete.
- A shared triad skill can resolve declared route pressure for multiple accepted runtime-organ standards, but it cannot erase per-organ validator, receipt, or source-body gaps.
- Runtime organ refinement must not collapse active residuals into green projection counts.
