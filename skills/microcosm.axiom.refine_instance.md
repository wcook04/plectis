## Authority Boundary

This skill source governs bounded axiom-instance correction only. It does not authorize source-of-record migration, new active axiom creation, or blanket downstream wiring.

Anti-claims:

- Refining an axiom row is not proof that its downstream principles are fully governed.
- Anti-principle guard resolution is not positive support for the axiom.
- Generated health can report the gap, but source evidence or validator computation must close it.
