## Authority Boundary

This skill source governs bounded standard-instance correction only. It does not authorize release, private source-body exposure, validator completeness claims, source-of-record migration, or broad doctrine wiring without source evidence.

Anti-claims:

- Count movement is not proof that every standard has complete validators, receipts, source coupling, and downstream skill uptake.
- Generated standard parity or graph reachability is not evidence of schema correctness beyond the checks actually run.
- Generated health can expose the gap, but JSON source evidence, validator computation, or a typed residual must settle it.
