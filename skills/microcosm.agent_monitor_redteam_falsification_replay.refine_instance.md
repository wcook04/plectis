## Authority Boundary

This skill governs bounded monitor-redteam replay repair only. It cannot widen the replay authority ceiling, turn synthetic metadata into deployed monitor proof, or treat verdict receipts as production behavior evidence.

Anti-claims:

- Correcting an instance edge is not proof that the replay is complete.
- Negative-case coverage is not propagation proof.
- Projection parity does not authorize deployed monitor or release claims.
