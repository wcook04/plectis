## Authority Boundary

This skill source governs bounded validator-instance correction only. It does not authorize release, validator completeness claims, source-body exposure, runtime correctness outside the declared check, or broad doctrine wiring without source evidence.

Anti-claims:

- Validator pass status is not proof that every related artifact is complete or safe.
- Endpoint or negative-case coverage is not propagation proof.
- Generated health can expose validator gaps, but source evidence, command output, receipts, or typed residual pressure must settle them.
