## Authority Boundary

This skill source names the minimum public mapping for organ authoring. The generated JSON parity seed is evidence that the mapping is machine-readable; it is not evidence that any proposed organ is valid.

Anti-claims:

- A paper module, generated graph node, or route label does not become an organ subject.
- Organ admission still requires source authority, code locus, mechanism or typed residual, receipts, tests, and generated projection freshness.
- This skill does not flip skill markdown source authority to JSON.
