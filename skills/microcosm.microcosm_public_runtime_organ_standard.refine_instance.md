## Authority Boundary

This skill source governs bounded instance correction for public runtime organ standards only. It does not authorize release, provider calls, live repository mutation, private body export, hidden fixture disclosure, benchmark scores, or whole-standard correctness claims.

Anti-claims:

- Correcting a source ref, receipt ref, or authority ceiling is not evidence that an organ is complete.
- A shared triad skill can resolve declared route pressure for multiple standards, but it cannot erase per-standard validator or receipt gaps.
- Public runtime standard refinement must not collapse active residuals into green projection counts.
