## Authority Boundary

This skill governs bounded route-observability instance repair only. It cannot widen the runtime authority ceiling, turn public route traces into live-session evidence, or treat receipts as provider execution proof.

Anti-claims:

- Correcting an instance edge is not proof that the replay is complete.
- Negative-case coverage is not propagation proof.
- Projection parity does not authorize live session, provider, or release claims.
