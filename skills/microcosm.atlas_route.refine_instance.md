## Authority Boundary

This skill governs bounded Atlas-route instance repair only. It cannot treat route projection freshness as source authority, release readiness, or private evidence access.

Anti-claims:

- Correcting a route edge is not proof that navigation is complete.
- Negative-case route coverage is not propagation proof.
- Projection parity does not authorize release claims.
