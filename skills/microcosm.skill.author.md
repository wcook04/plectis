## Authority Boundary

This skill source governs skill admission only. Generated JSON parity, resolved standard-triad edges, graph reachability, and skill catalog visibility are navigability evidence; they do not prove agent uptake, runtime correctness, release readiness, or a source-of-record migration away from Markdown.

Anti-claims:

- A skill heading or broad prose note is not enough to infer operated standard, acted-on kind, concept, or mechanism neighbours.
- Skill JSON parity is not evidence that the procedure is complete, used correctly by agents, or safe for private/provider material.
- This self-applying route must not weaken std_microcosm_skill merely to make triad counts greener.
