## Authority Boundary

This skill governs bounded Batch 10 live-source drift instance repair only. It cannot widen the authority ceiling into route authority, ledger mutation, source mutation, mission-transaction execution, git operation authority, publication, release authority, or private runtime export.

Anti-claims:

- Corrected digest rows are not source mutation authority.
- Body-free receipts do not prove live route correctness.
- Projection health must not hide ledger or git-operation residuals.
