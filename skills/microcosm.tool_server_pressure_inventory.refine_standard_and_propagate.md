## Authority Boundary

This skill governs class-level tool-server pressure inventory propagation only. It does not authorize live process reads, process signalling, host mutation, provider calls, release, private-data equivalence, or whole-system correctness.

Anti-claims:

- Standard propagation cannot turn synthetic fixture validation into live host authority.
- Generated health improvement is not process-safety proof.
- Receipt existence cannot erase per-instance source or redaction gaps.
