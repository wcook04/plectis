## Authority Boundary

- A validator name is not automatically a mechanism.
- A mechanism without source row, code grounding or residual, and host-organ routing remains pressure, not proof.
- This skill does not claim runtime correctness, propagation proof, release authority, or JSON source-of-record migration.
