## Authority Boundary

This skill governs bounded agent-trace instance repair only. It cannot widen the trace authority ceiling or treat route evidence as private transcript access.

Anti-claims:

- Fixing trace metadata is not proof of trace completeness.
- Negative-case trace coverage is not propagation proof.
- Projection parity does not authorize release or provider claims.
