## Authority Boundary

This skill source governs bounded paper-module-instance correction only. It does not authorize source-of-record migration, new subject fabrication, release authority, or broad doctrine wiring without source evidence.

Anti-claims:

- Count movement is not proof that every paper module has complete concept, law, code, and receipt neighbours.
- Endpoint coverage, generated JSON parity, or Mermaid availability is not propagation proof.
- Generated health can expose the gap, but capsule/source evidence or validator computation must close it.
