## Authority Boundary

This skill governs bounded sabotage/scheming replay repair only. It cannot widen the replay authority ceiling, turn synthetic metadata into deployed monitor proof, or treat receipts as private reasoning or exploit evidence.

Anti-claims:

- Correcting an instance edge is not proof that the replay is complete.
- Negative-case coverage is not propagation proof.
- Projection parity does not authorize deployed monitor or release claims.
