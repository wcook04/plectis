## Authority Boundary

This skill governs bounded memory-conflict replay repair only. It cannot widen the replay authority ceiling, turn fixtures into live memory evidence, or treat receipts as production behavior proof.

Anti-claims:

- Correcting an instance edge is not proof that the replay is complete.
- Negative-case coverage is not propagation proof.
- Projection parity does not authorize live memory or release claims.
