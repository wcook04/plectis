# Formal Math Lean Proof Witness

## Teleology

`formal_math_lean_proof_witness` is a deferred compatibility boundary. The
accepted public runtime spine can describe where a future Lean witness would
attach, but this root does not currently execute Lean or Lake.

## Public Contract

The current contract is negative: Lean/Lake remains unauthorized, proof bodies
remain toxic unless synthetic and redacted, and proof-spine diagnostics remain
evidence cells rather than formal proof authority.

## Receipt Expectations

Expected Lean receipts are intentionally absent until a future reducer
authorizes a tiny Lean witness slice. `receipts/preflight/dependency_preflight.json`
records the deferral boundary.

## Anti-Claim

This module is a deferred-boundary paper module only. It does not authorize
Lean/Lake execution, formal proof authority, runtime correctness, public release
readiness, hosted-public readiness, publication, recipient work, provider calls,
private-data equivalence, or whole-system correctness.
